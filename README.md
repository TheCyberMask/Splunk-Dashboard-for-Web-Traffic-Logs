# Splunk-Dashboard-for-Web-Traffic-Logs
A Splunk dashboard that visualizes webserver traffic, showing total requests, successful responses, client errors, top URLs, and traffic by IP address to help monitor website performance and detect abnormal activity

OBJECTIVE

The objective of this project is to analyze and monitor web server traffic using Splunk by visualizing total web requests, successful responses, client errors, top requested URLs, and web traffic by client IP address to improve performance and enhance security visibility.


TOOLS USED 

 .  Splunk
 .  apache_logs.json file

SKILLS LEARNED

. Log analysis using Splunk

. Creating and customizing Splunk dashboards

. Understanding HTTP status codes (2xx, 4xx, 5xx)

. Identifying traffic patterns and user behaviors

. Detecting abnormal or suspicious IP activity

. Working with Splunk search queries (SPL)

. Visualizing data through charts and panels
